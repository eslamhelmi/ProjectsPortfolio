Excel

Microsoft Excel is one of the most popular applications for data analysis. Equipped with built-in power query, pivot tables, charts and other statistical functions making it without a doubt the most used analytic tool available. It is an all-in-one data management software that allows you to easily import, explore, clean, analyze, and visualize your data

Sample Projects

This section includes sample projects using excel analyzing different topics in several business fields. Each project is explained and discussed with a step-by-step guide to address the problem and outline the techniques used to clear it up

Account Sales Analysis - A New Era of Retail Has Begun

A project that analyzes different acounts sales data provided from JPMorgan Chase & Co. The project aims to evaluate the performance of different accounts and how several marketing methods can affect the sales volume. It also addresses the impact of selling down some products with other ones and how CAGR is increasing differently according to the account type.
