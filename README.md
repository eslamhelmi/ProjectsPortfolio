# Islam Helmy Portfolio Website

Professional portfolio built with the HTML5 UP Editorial template, showcasing forecasting, supply chain analytics, and business intelligence projects.

## 🚀 Quick Setup

### Step 1: Download the Editorial Template Assets

The HTML files are ready, but you need CSS/JS from HTML5 UP:

1. Go to: **https://html5up.net/editorial/download**
2. Download and extract the ZIP
3. Copy these folders into your portfolio folder:
   - `assets/` (contains css, js, webfonts)
   
### Step 2: Upload to GitHub

Using GitHub Desktop:
1. Clone your repository (ProjectsPortfolio)
2. Delete old files (keep .git folder)
3. Copy all new files into the folder
4. Commit and Push

## ✏️ Customization Guide

### To Edit Text Content
1. Open any `.html` file in VS Code
2. Find and replace text
3. Save and refresh browser

### To Add a New Case Study
1. Copy `case-studies/cmaf-task1.html`
2. Rename (e.g., `new-project.html`)
3. Edit content
4. Add link in `case-studies.html` and sidebar menus

### To Add a Blog Post
1. Create new HTML file in `blog/` folder (create folder if needed)
2. Use case study structure as template
3. Add link in `blog.html`
