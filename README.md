# Islam Helmy Portfolio Website

Professional portfolio built with the HTML5 UP Editorial template, showcasing forecasting, supply chain analytics, and business intelligence projects.

## 🚀 Quick Setup

### Step 1: Download the Editorial Template Assets

The HTML files are ready, but you need CSS/JS from HTML5 UP:

1. Go to: **https://html5up.net/editorial/download**
2. Download and extract the ZIP
3. Copy the `assets/` folder into your portfolio folder

### Step 2: Upload to GitHub

Using GitHub Desktop:
1. Open GitHub Desktop
2. Navigate to your cloned ProjectsPortfolio folder
3. Delete old files (keep .git folder)
4. Copy all new files into the folder
5. Commit and Push

## 📸 Replacing Images

### Profile Photo
Replace `images/profile.svg` with your professional photo:
- Recommended size: 400x400px
- Format: JPG or PNG
- Name it `profile.jpg` and update index.html reference

### Certificate (Optional)
Add your IIF certificate as `images/iif-certificate.jpg` if you want to link to it.
