SQL [Structure Query Language]

SQL stands for Structured Query Language. It is the standard language for relational database management systems. It is especially useful in handling organized data comprised of entities (variables) and relations between different entities of the data, in other words, retrieving and manipulating structured databases

Sample Projects

This section includes sample projects using SQL analyzing different databases in real life. Each project is explained and discussed with a step-by-step guide to address the data we want to be retreived and outline the techniques used to clear it up. I have used three different relational database managment systems; MySQL, PostgreSQL and Microsoft SQL Server
